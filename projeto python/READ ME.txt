------------IMPORTANTE------------

Antes de iniciar o programa certifique-se que o mySQL está up e localizado na porta 3306

O usuario do banco deve ser "root" e a senha deve estar vazia

----------------------------------

Caso queira executar o app apartir do código fonte é necessario instalar o kivyMD e o mySQL connector

comandos:

py -m pip install https://github.com/kivymd/KivyMD/archive/master.zip
py -m pip install mysql.connector
